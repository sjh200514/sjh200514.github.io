数字信号处理复习总结

01 信号的时域分析

一、信号的分类

  1. 周期信号的 x(t), y(t) 的周期为T1和T2，若周期之比 T1/T2 为有理数，则其和信号 x(t) + y(t) 仍然是周期信号，其周期为 T1 和 T2 的最小公倍数。

  2. 周期信号可能分解为周期信号, 非周期信号也可能分解为周期信号

  3. 连续正弦信号一定是周期信号,而正弦序列不一定是周期序列。两连续周期信号之和不一定是周期信号,而两周期序列之和一定是周期序列。

  4. 能量信号与功率信号

连续信号 离散信号
能量 $W=\lim\limits_{T\rightarrow\infty}\int_{-T}^{T}|x(t)|^2dt$ $W=\lim\limits_{N\rightarrow\infty}\sum\limits_{n=-N}^{N}|x[n]|^2$
功率 $P=\lim\limits_{T\rightarrow\infty}\frac{1}{2T}\int_{-T}^{T}|x(t)|^2dt$ $W=\lim\limits_{N\rightarrow\infty}\frac{1}{2N+1}\sum\limits_{n=-N}^{N}|x[n]|^2$
  • 能量信号：0 < W < ∞, P = 0。（信号的能量有限）
  • 功率信号：W → ∞, 0 < P < ∞。（信号的能量无限，但其功率是有限的）
  • 直流信号与周期信号都是功率信号

  1. 信号 x(t) 可以是一个既非功率信号,又非能量信号。但一个信号不可能同时既是功率信号, 又是能量信号。周期信号都是功率信号；非周期信号可能是能量信号(t → ∞, x(t) = 0)也可能是功率信号(t → ∞, x(t) ≠ 0)。

  2. 因果信号：当 t < 0 时 x(t) = 0，当 t > 0 时 x(t) ≠ 0。

二、典型的信号

  1. 虚指数信号 x(t) = Aejω0t
  • 周期性：x(t) = x(t + T0) = ejω0t = ejω0(t + T0)
  • 若 ω0 ≠ 0，基波周期满足 ω0T0 = 2πk，即 $T_0=\frac{2\pi}{|\omega_0|}$
  • ω0 越大，信号震荡速率越高
  1. 复指数信号 x(t) = Aest
  • A 使用极坐标表示：A = |A|ejθ
  • s 使用笛卡尔坐标表示：s = r + jω0，x(t) = Aest = |A|ertej(ω0t + θ)
  • x(t) = Aest = |A|ertej(ω0t + θ) = |A|ertcos (ω0t + θ) + j|A|ertsin (ω0t + θ)，具有指数衰减振幅的正弦信号常称为阻尼正弦振荡。
  1. 抽象信号 $Sa(t)=\frac{\sin t}{t}$

  • $Sa(0)=\lim\limits_{t\rightarrow0}\frac{\sin t}{t}=1,Sa(k\pi)=0$

  • $\int_{-\infty}^{\infty}Sa(t)dt=\pi,\int_{0}^{\infty}Sa(t)dt=\frac{\pi}{2}$

  • $\int_{0}^{\infty}\frac{\sin(t)dt}{t}=\int_{0}^{\infty}\sin(t)dt\int_{0}^{\infty}e^{-tx}dx=\int_{0}^{\infty}\int_{0}^{\infty}\sin(t)e^{-tx}dtdx$

    $I=\int_{0}^{\infty}\sin(t)e^{-tx}dt=-\frac{1}{x}\int_{0}^{\infty}\sin(t)d(e^{-tx})=-\frac{1}{x}\sin(t)e^{-tx}\big|_{t=0}^{\infty}+\frac{1}{x}\int_{0}^{\infty}\cos(t)e^{-tx}dt=\frac{1}{x}\int_{0}^{\infty}\cos(t)e^{-tx}dt$

    $=\frac{1}{x}\int_{0}^{\infty}\cos(t)e^{-tx}dt=-\frac{1}{x^2}\int_{0}^{\infty}\cos(t)d(e^{-tx})=-\frac{1}{x^2}\cos(t)e^{-tx}\big|_{t=0}^{\infty}+\frac{1}{x^2}\int_{0}^{\infty}e^{-tx}d(\cos(t))$

    $=\frac{1}{x^2}-\frac{1}{x^2}I$

    解得： $I=\frac{1}{1+x^2}$

    $\int_{0}^{\infty}\frac{\sin(t)}{t}=\int_{0}^{\infty}\int_{0}^{\infty}\sin(t)e^{-tx}dtdx=\int_{0}^{\infty}\frac{1}{1+x^2}dx=\arctan(x)\big|_{x=0}^{\infty}=\frac{1}{2}$

  • $sinc(t)=\frac{\sin ct}{ct}$

  1. 虚指数序列：x[n] = ejω0n

  • ejω0n 周期为 2π，不具有随 ω0 增加而增加震荡速率的性质

  • 周期性：x[n + N] = x[n] = ejω0n = ejω0nejω0N，满足 ω0N = m2π, m ∈ ℕ+

  • 连续信号 ejω0t 离散时间序列信号 ejω0n
    ω0 不同,信号不同 ω0 相差 2π 整倍数,信号相同
    任何 ω0 信号均为周期信号 满足 ω0N = m2π, m ∈ ℕ+ 才是周期信号
    基波周期：$\begin{cases} \omega_0=0, & 无定义 \\ \omega_0\ne0, & T_0=\frac{2\pi}{|\omega_0|} \end{cases}$ 基波周期：$\begin{cases} \omega_0=0, & 无定义 \\ \omega_0\ne0, & T_0=\frac{2\pi m}{|\omega_0|} \end{cases}$
  1. 单位脉冲序列：$\delta[n]=\begin{cases} 1, &n=0\\ 0, &n\ne 0\end{cases}$
  • x[n]δ[n] = x[0]δ[n], x[n]δ[n − n0] = x[n0]δ[n − n0]
  • 使用单位脉冲表示任意离散时间信号
  1. 单位脉冲序列：$u[n]=\begin{cases} 1, &n\ge0\\ 0, &n< 0\end{cases}$
  • u[n] 可看作延时脉冲的叠加：$u[n]=\sum\limits_{k=0}^{\infty}\delta[n-k]$
  • 单位阶跃序列和单位脉冲序列的相互表示：$\delta[n]=u[n]-u[n-1],u[n]=\sum\limits_{k=-\infty}^{n}\delta[k]$
  1. 矩形序列：$r[n]=\begin{cases} 1, &0\le n\le N-1\\ 0, &o.w.\end{cases},r[n]=u[n]-u[n-N]=\sum\limits_{m=0}^{N-1}\delta[n-m]$
  2. 斜变序列：$x[n]=nu[n]=\sum\limits_{k=0}^{\infty}k\delta[n-k]$
  3. 奇异信号：信号本身或其导数具有不连续点(跳变)，例如单位斜变信号 $r(t)=\begin{cases}t,&t\ge0\\0,&t<0\end{cases}$
  4. 单位阶跃信号：$u(t)==\begin{cases}1,&t>0\\0,&t<0\end{cases}$
  5. 冲激信号 δ(t)
  • 冲激信号的狄拉克定义：δ(t) = 0(t ≠ 0); ∫∞∞δ(t)dt = 1
  • 抽样（筛选）特性：x(t)δ(t − t0) = x(t0)δ(t − t0)
  • ∫−∞+∞x(t)δ(t − t0)dt = ∫−∞+∞x(t0)δ(t − t0)dt = x(t0)∫−∞+∞δ(t − t0)dt = x(t0)
  • x(t) = ∫−∞+∞x(τ)δ(τ − t)dτ = ∫−∞+∞x(τ)δ(t − τ)dτ = x(t) * δ(t)
  • $\delta(\alpha t)=\frac{1}{|\alpha|}\delta(t)(\alpha\ne0)$，α = −1 有 δ(−t) = δ(t)，因此 δ(t) 为偶函数
  • $\delta(at+b)=\frac{1}{|a|}\delta(t+\frac{b}{a})$

三、信号的分解

  1. 基于冲激信号进行信号分解：$x(t)=\int_{-\infty}^{\infty}x(\tau)\delta(t-\tau)d\tau,x[n]=\sum\limits_{k=-\infty}^{\infty}x[k]\delta[n-k]$

  2. 若 n 个函数 g1(t), g2(t), ..., gn(t) 构成函数集，且在区间 (t1, t2) 满足 $\begin{cases} <g_i(t),g_j(t)>=0,i\ne j\\<g_i(t),g_i(t)>=K_i^2\end{cases}$，则函数集为正交函数集。 重构函数 $x(t)=\sum\limits_{r=1}^{n}c_rg_r(t),\overline{\epsilon^2}=\frac{1}{t_2-t_1}\int_{t_1}^{t_2}[x(t)-\sum\limits_{r=1}^{n}c_rg_r(t)]^2dt,其中c_r=\frac{\int_{t_1}^{t_2}x(t)g_r(t)dt}{\int_{t_1}^{t_2}g_r^2(t)dt}=\frac{<x(t),g_r(t)>}{K_r^2}$

  3. 若 n 个函数 g1(t), g2(t), ..., gn(t) 构成正交函数集，不存在 ℊ(t) 满足 0 < ∫−∞∞ℊ2(t)dt < ∞ 且  < ℊ(t), gr(t) >  = 0, r = 1, 2, ..., n，则 g1(t), g2(t), ..., gn(t) 构成完备正交函数集。

  4. 帕塞瓦尔定理：$\int_{t_1}^{t_2}x^2(t)dt=\sum_\limits{r}^{\infty}c^2_r$， 信号所含有的功率恒等于此信号在完备正交函数集中各分量功率的总和。

  5. 冲激偶信号 $\delta^{'}(t)=\frac{d\delta(t)}{dt}$

  • ∫−∞∞δ′(t)dt = 0

  • δ′(−t) = −δ′(t)

  • 筛选特性：x(t)δ′(t − t0) = x(t0)δ′(t − t0) − x′(t0)δ(t − t0) x(t)δ(t − t0) = x(t0)δ(t − t0) 两边对 t 求导可得：x′(t)δ(t − t0) + x(t)δ′(t − t0) = x(t0)δ′(t − t0) x(t)δ′(t − t0) = x(t0)δ′(t − t0) − x′(t)δ(t − t0) = x(t0)δ′(t − t0) − x′(t0)δ(t − t0)

  • 抽样特性：∫−∞∞x(t)δ′(t − t0)dt = −x′(t0)

    ∫−∞∞x(t)δ′(t − t0)dt = ∫−∞∞[x(t0)δ′(t − t0) − x′(t0)δ(t − t0)]dt  = x(t0)∫−∞∞δ′(t − t0)dt − x′(t0)∫−∞∞δ(t − t0)dt = 0 − x′(t0) = −x′(t0)

  • 展缩特性：$\delta^{'}(\alpha t)=\frac{1}{\alpha|\alpha|}\delta^{'}(t)(\alpha\ne0),\delta^{'}(at+b)=\frac{1}{a|a|}\delta^{'}(t+\frac{b}{a})(a\ne0)$

  • x(t) * δ′(t) = x′(t), x(t) * δ′(t − t0) = x′(t − t0) x(t) * δ′(t − t0) = ∫−∞+∞x(k)δ′(t − t0 − k)dk = −∫−∞+∞x(k)δ′(k − (t − t0))dk  = −∫−∞+∞[x(t − t0)δ′(k − (t − t0)) − x′(t − t0)δ(k − (t − t0))]dk  = −x(t − t0)∫−∞+∞δ′(k − (t − t0)) + x′(t − t0)∫−∞+∞δ(k − (t − t0))dk  = 0 + x′(t − t0) = x′(t − t0)

02 系统的时域分析

一、系统的分类和描述

  1. 系统的输入输出描述：N 阶微分方程或 N 阶差分方程，如 $L\frac{di(t)}{dt}+Ri(t)=x(t)$
  2. 系统的状态空间描述包括连续时间系统和离散时间系统
  3. 连续时间系统
  • 系统的输入激励与输出响应都必须为连续时间信号
  • 连续时间系统的数学模型是微分方程
  1. 离散时间系统
  • 系统的输入激励与输出响应都必须为离散时间信号
  • 离散时间系统的数学模型是差分方程
  • 优点：可靠性高，随环境变化小；系统参数精度高； 利用存储器存储信息；应用更灵活；易消除噪声，易处理低频信号；多维信号处理技术成熟
  1. 系统互联的方式：串联（级联）、并联和反馈连接
  2. 记忆系统和无记忆系统
  • 无记忆系统：系统的输出只取决于系统该时刻的输入
  • 恒等系统：y(t) = x(t); y[n] = x[n]
  • 累加器：$y[n]=\sum\limits_{k=-\infty}^{n}x[k]$
  • 延时单元：y[n] = x[n − 1]
  1. 可逆系统
  • 系统在不同的输入下，导致不同的输出，则系统可逆
  • 如果一个系统可逆，则存在一个逆系统，系统与逆系统级联，作用等效于恒等系统
  1. 因果系统
  • 当且仅当输入信号激励系统时才产生系统输出响应的系统
  • 如果响应 y(t) 并不依赖于将来的激励如 x(t + 1)，那么系统就是因果的
  1. 稳定系统
  • 稳定系统在微小的输入下的响应不会发散
  • 指有界输入产生有界输出的系统
  1. 线性系统
  • 如果一个输入信号是由几个信号加权组成的，则线性系统的输出是系统对这组信号中每一个响应的加权和
  • 齐次性（比例性）：若 x1(t) → y1(t)，则 Kx1(t) → Ky1(t)
  • 可加性（叠加性）：若 x1(t) → y1(t), x2(t) → y2(t)，则 x1(t) + x2(t) → y1(t) + y2(t)
  • 连续线性特性：若 x1(t) → y1(t), x2(t) → y2(t)，则 $\alpha x_1(t)+\beta x_2(t)\rightarrow \alpha y_1(t)+\beta y_2(t)(\alpha,\beta为任意常数)$
  • 离散线性特性：若 x1[n] → y1[n], x2[n] → y2[n]，则 $\alpha x_1[n]+\beta x_2[n]\rightarrow \alpha y_1[n]+\beta y_2[n](\alpha,\beta为任意常数)$
  • 系统的齐次性和可加性可以不同时满足 y(t) = Re[x(t)] 满足可加性，不满足齐次性 $y(t)=\frac{[x^{'}(t)]^2}{x[t]}$ 满足齐次性，不满足可加性
  1. 时变系统和时不变系统
  • 系统的输出响应与输入激励的关系不随输入激励作用于系统的时间起点而改变，就称为时不变系统。否则，就称为时变系统。
  • 线性时不变系统：Linear Time Invariant (LTI)
  • 线性时不变系统可由定常系数的线性微分方程或差分方程描述。
  • 连续时不变系统：x(t) → y(t) 则 x(t − t0) → y(t − t0)
  • 离散时不变系统：x[n] → y[n] 则 x[n − k] → y[n − k]

二、卷积

  1. 线性系统的单位脉冲响应
  • $x[n]=\sum\limits_{k=-\infty}^{+\infty}x[k]\delta[n-k]$
  • $y[n]=\sum\limits_{k=-\infty}^{+\infty}x[k]h_k[n]$，其中 hk[n] 为单位脉冲 δ[n − k] 的响应
  1. 卷积和

  • 卷积和定义：$y[n]=\sum\limits_{k=-\infty}^{+\infty}x[k]h[n-k]$

  • 符号记为：y[n] = x[n] * h[n]

  • 计算 x[n] = αnu[n] 与 h[n] = βn 的卷积和 $x[n]*h[n]=\alpha^nu[n]*\beta^nu[n]=\sum\limits_{k=-\infty}^{+\infty}\alpha^ku[k]\beta^{n-k}u[n-k]$ $=\begin{cases}\sum\limits_{k=0}^{n}\alpha^k\beta^{n-k},&n\ge0\\0,&n<0\end{cases}=\begin{cases}\frac{\beta^{n+1}-\alpha^{n+1}}{\beta-\alpha}u[n],&\alpha\ne\beta\\(n+1)\alpha^nu[n],&\alpha=\beta\end{cases}$

  • 表格法 设 x[n] 和 h[n] 都是因果序列，则有 $y[n]=x[n]*h[n]=\sum\limits_{k=0}^{n}x[k]h[n-k],n\ge0$ 将 h[n] 的值顺序排成一行，将 x[n] 的值顺序排成一列，行与列的交叉点记入相应 x[n] 与 h[n] 的乘积：

  1. 卷积积分
  • y(t) = x(t) * h(t) = ∫−∞∞x(τ)h(t − τ)dτ
  • 已知 x(t) = e−tu(t), h(t) = u(t)，计算 x(t) * h(t) x(t) * h(t) = ∫−∞∞e−τu(τ)u(t − τ)dτ = ∫0te−τdτ = −e−τ|τ = 0t = 1 − e−t

三、卷积的性质和特殊信号的卷积

  1. 卷积的性质

  • 交换律：x1(t) * x2(t) = x2(t) * x1(t)

  • 分配律：[x1(t) + x2(t)] * x3(t) = x1(t) * x3(t) + x2(t) * x3(t)

  • 结合律：[x1(t) * x2(t)] * x3(t) = x1(t) * [x2(t) * x3(t)]

  • 平移特性：已知 x1(t) * x2(t) = y(t)，则 x1(t − t1) * x2(t − t2) = y(t − t1 − t2) $x_1(t-t_1)*x_2(t-t_2)=\int_{-\infty}^{\infty}x_1(\tau-t_1)x_2(t-t_2-\tau)d\tau\overset{\tau-t_1=\lambda}{=}\int_{-\infty}^{\infty}x_1(\lambda)x_2(t-t_1-t_2-\lambda)d\lambda=y(t-t_1-t_2)$

  • 展缩特性：已知 x1(t) * x2(t) = y(t)，则 $x_1(\alpha t)*x_2(\alpha t)=\frac{1}{|\alpha|}y(\alpha t)$ $x_1(\alpha t)x_2(\alpha t)=\int_{-\infty}^{\infty}x_1(\alpha\tau)x_2(\alpha t-\alpha\tau)d\tau\overset{\alpha\tau=\lambda}{=}\frac{1}{|\alpha|}\int_{-\infty}^{\infty}x_1(\lambda)x_2(\alpha t-\lambda)d\lambda=\frac{1}{|\alpha|}y(\alpha t)$

  • 微分特性：$\frac{d}{dt}[x_1(t)*x_2(t)]=x_1(t)*\frac{d}{dt}x_2(t)=\frac{d}{dt}x_1(t)*x_2(t)$

    $((f(\tau)*g(\tau))(t))^{'}=\frac{d}{dt}(\int_{-\infty}^{\infty}f(\tau)g(t-\tau)d\tau)=(f(\tau)*g^{'}(\tau))(t)$ 由 Leibniz 求导法则： $\frac{d}{dx}(\int_{a(x)}^{b(x)}f(x,t)dt)=f(x,b(x))\cdot\frac{d}{dx}b(x)-f(x,a(x))\cdot\frac{d}{dx}a(x)+\int_{a(x)}^{b(x)}\frac{\partial}{\partial x}f(x,t)dt$ $\frac{d}{dt}(\int_{-\infty}^{\infty}f(\tau)g(t-\tau)d\tau)=\int_{-\infty}^{\infty}\frac{\partial}{\partial t}(f(\tau)g(t-\tau))d\tau=\int_{-\infty}^{\infty}f(\tau)g^{'}(\tau)d\tau=(f(\tau)*g^{'}(\tau))(t)$

  • ∫−∞tf(τ)dτ = ∫−∞∞f(τ)u(t − τ)dτ = f(t) * u(t)

  • 积分特性：∫−∞t[x1(τ) * x2(τ)]dτ = x1(t) * ∫−∞t[x2(τ)]dτ = ∫−∞t[x1(τ)]dτ * x2(t) ∫−∞t(f(ξ) * g(ξ))(τ)dτ = ((∫−∞τf(ξ)dξ) * g(τ))(t) ((∫−∞τf(ξ)dξ) * g(τ))(t) = ((f(ξ) * u(ξ))(τ) * g(τ))(t) = ((f(ξ) * g(ξ))(τ) * u(τ))(t) = ∫∞t(f(ξ) * g(ξ))(τ)dτ

  • 等效特性：y(t) = x1(t) * x2(t), y(i)(t) = x1(j)(t) * x2(i − j)(t)

  1. 奇异信号的卷积
  • δ(t) * δ(t) = δ(t)
  • x(t) * δ(t − T) = x(T)
  • x(t) * δ′(t) = x′(t)
  • u(t) * u(t) = ∫−∞tu(τ)dτ = tu(t)
  • x(t) * u(t) = ∫−∞tx(τ)dτ = x(−1)(t)(积分) x(t) * δ(t) = x(t)(直通) x(t) * δ(t − t0) = x(t0)(延迟)
  • x1(t) * x2(t) = x1(−1)(t) * x2′(t) = x1′(t) * x2(−1)(t) = [x1′(t) * x2(t)](−1)
  1. 卷积和的性质
  • 交换律：x[n] * h[n] = h[n] * x[n]
  • 结合律：x[n] * {h1[n] * h2[n]} = {x[n] * h1[n]} * h2[n]
  • 分配律：x[n] * {h1[n] + h2[n]} = x[n] * h1[n] + x[n] * h2[n]
  • 位移特性：x[n] * δ[n − k] = x[n − k] 若 x[n] * h[n] = y[n]，则 x[n − k] * h[n − l] = y[n − (k + l)]
  • 差分特性：若 x[n] * h[n] = y[n]，则 ∇x[n] * h[n] = x[n] * ∇h[n] = ∇y[n]
  • 求和特性：若 x[n] * h[n] = y[n]，则 $x[n]*\sum\limits_{k=-\infty}^{n}h[k]=(\sum\limits_{k=-\infty}^{n}x[k])*h[n]=\sum\limits_{k=-\infty}^{n}y[k]$

四、卷积的应用

  1. 级联系统的冲激响应等于两个子系统冲激响应的卷积： y(t) = x(t) * h1(t) * h2(t) = x(t) * (h1(t) * h2(t)) 交换两个级联系统的先后连接次序不影响系统总的冲激响应。
  2. 并联系统的冲激响应等于两个子系统冲激响应之和：y(t) = y1(t) + y2(t) = x(t) * h1(t) + x(t) * h2(t) = x(t) * (h1(t) + h2(t))
  3. LTI系统因果的充分必要条件
  • 因果连续时间LTI系统的冲激响应必须满足：h(t) = 0, t < 0
  • 因果离散时间LTI系统的单位脉冲响应必须满足：h[n] = 0, n < 0
  1. LTI系统稳定的充分必要条件
  • 连续时间LTI系统稳定的充分必要条件是：∫−∞∞|h(τ)|dτ < ∞
  • 离散时间LTI系统稳定的充分必要条件是：$\sum\limits_{k=-\infty}^{\infty}|h[k]|<\infty$

  1. LTI特性与单位冲激响应的关系

    特性 连续时间系统 离散时间系统
    无记忆性 h(t) = cδ(t) h[n] = cδ[n]
    因果性 h(t) = 0, t < 0 h[n] = 0, n < 0
    稳定性 ∫−∞∞|h(τ)|dτ < ∞ $\sum\limits_{k=-\infty}^{\infty}|h[k]|<\infty$
    可逆性 h0(t) * h−(t) = δ(t) h0[n] * h−[n] = δ[n]
    时不变性 δ(t − t0) → h(t − t0) δ[n − n0] → h[n − n0]
    线性 αδ(t) → αh(t) αδ[n] → αh[n]

  2. 相关运算：衡量信号之间的相似程度

  • x(t) ⋆ h(t) = ∫−∞∞x(τ)h(τ + t)dτ = ∫−∞∞x(τ − t)h(τ)dτ
  • 注意不满足 x(t) ⋆ h(t) = h(t) ⋆ x(t)
  • x(t) ⋆ h(t) = x(−t) * h(t)
  1. 通过内积  < f, g >  = ∫−∞∞f(t)g(t)dt 刻画两个信号的相似性

03 系统的时域分析

  1. 对于 n 时刻的输出仅仅和 n 之前时刻的输入相关的模型，称为MA (Moving Average)模型。 示例：空谷回声 x[n] 为当前时刻发出的声音，y[n] 为当前时刻听到的声音，bk 刻画声音传播过程中的衰减 y[n] = x[n] + bkx[n − k]

  2. 对于 n 时刻的输出仅仅和 n 之前时刻的输出相关的模型，称为AR (Auto Regressive)模型。 示例：银行理财 当前账户总额为 y[n]，ai 为存款利率 y[n] = y[n − 1] + aiy[n − 1]

  3. 一个线性时不变(LTI)系统可以用常系数微分方程或常系数差分方程描述，但一个常系数微分方程或差分方程描述的系统不一定是线性时不变系统。

  4. 微分方程求解 微分方程 y(N)(t) + aN − 1y(N − 1)(t) + ... + a1y′(t) + a0y(t) = bMx(M)(t) + bM − 1(M − 1)(t) + ... + b1x′(t) + b0x(t) 的（完全）解由齐次解 yh(t) 和特解 yp(t) 构成。

    构建特征方程 αN + aN − 1αN − 1 + ... + a1α + a0 = 0，求解 N 个特征根 α1, ..., αN，分情况讨论

  • 特征根各不相同（无重根），齐次解为 $y_h(t)=A_1e^{\alpha_1t}+A_2e^{\alpha_2t}+...+A_Ne^{\alpha_Nt}=\sum\limits_{i=1}^{N}A_ie^{\alpha_it}$

  • 若有共轭复根 α1 = m + jn, α2 = m − jn，则 yh(t) = A1eα1t + A2eα2t = emt((A1 + A2)cos nt + (A1 − A2)jsin nt)，齐次解为增长或衰减的正弦函数

  • 特征根有重根，设 α1 为 k 阶重根，齐次解为 $y_h(t)=(A_1t^{k-1}+A_2t^{k-2}+...+A_{k-1}t+A_k)e^{\alpha_1 t}+A_2e^{\alpha_2t}+...+A_{N-k+1}e^{\alpha_{N-k+1}t}=(\sum\limits_{i=1}^{k}A_it^{k-i})e^{\alpha_1 t}+\sum\limits_{i=2}^{N-k+1}A_ie^{\alpha_i t}$

    特解的形式和激励函数的形式有关，通过代入激励函数，观察形式，代入微分方程求解待定系数

$$ \begin{array}{|c|c|} \hline \text{激励函数 } x(t) & \text{响应函数 } y(t) \text{ 的特解} \\ \hline \text{常数} & B \\ \hline t^p & B_1t^p + B_2t^{p-1} + \dots + B_pt + B_{p+1} \\ \hline e^{\alpha t} & Be^{\alpha t} \\ \hline \cos(\omega t), \sin(\omega t) & B_1\cos(\omega t) + B_2\sin(\omega t) \\ \hline t^pe^{\alpha t}\cos(\omega t) & {\begin{aligned} & (B_1t^p + B_2t^{p-1} + \dots + B_pt + B_{p+1})e^{\alpha t}\cos(\omega t) \\ & + (D_1t^p + D_2t^{p-1} + \dots + D_pt + D_{p+1})e^{\alpha t}\sin(\omega t) \end{aligned}} \\ t^pe^{\alpha t}\sin(\omega t) & \\ \hline \end{array} $$

  • 若特解和齐次解重复，如均为 eαt，则特解设为 B0teαt

  • 若对应的特征解为二重根，齐次解为 teαt，特解为 B0t2eαt

  • 若特解为0时刻加入，则微分方程的解空间为 0+ ≤ t < ∞

    基于齐次解 yh(t) 和特解 yp(t) 构造完全解 y(t) = yh(t) + yp(t)，需借助边界条件确定齐次解中的待定系数，如 y(0), y′(0) 的值。
